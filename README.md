# 🍜 Cuisine Classification using Machine Learning

A machine learning project that predicts the cuisine type based on ingredient presence. This project uses classification techniques to identify cuisines such as Thai, Japanese, Chinese, Indian, and Korean from ingredient data.

## 📌 Project Overview

This project applies machine learning to classify cuisines using ingredient-based features. The dataset is analyzed, visualized, balanced using SMOTE, and trained using a Logistic Regression model for prediction.

The workflow includes:

- Data loading and preprocessing
- Cuisine-wise ingredient analysis
- Data visualization
- Feature engineering
- Class balancing using SMOTE
- Model training with Logistic Regression
- Prediction and evaluation

## 🚀 Features

- Predict cuisine type from ingredients
- Visualize cuisine distribution
- Analyze top ingredients for each cuisine
- Handle imbalanced data using SMOTE
- Train a machine learning classification model
- Generate prediction probabilities
- Evaluate model performance using classification reports

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Imbalanced-learn (SMOTE)
- Jupyter Notebook

## 📂 Dataset

The dataset contains ingredient information for different cuisines such as:

- Thai
- Japanese
- Chinese
- Indian
- Korean

Each cuisine is represented using ingredient presence features.

## ⚙️ Workflow

### 1. Data Preprocessing
- Load cuisine dataset
- Clean and prepare features
- Remove unnecessary columns

### 2. Exploratory Data Analysis
- Cuisine distribution visualization
- Top ingredients analysis for each cuisine

### 3. Feature Preparation
- Separate labels and features
- Remove common ingredients to improve classification quality

### 4. Class Imbalance Handling
- Apply SMOTE to balance cuisine classes

### 5. Model Training
- Train a Logistic Regression classifier
- Split data into training and testing sets

### 6. Prediction
- Predict cuisine type from ingredient combinations
- Display prediction probabilities

### 7. Evaluation
- Accuracy measurement
- Classification report generation
